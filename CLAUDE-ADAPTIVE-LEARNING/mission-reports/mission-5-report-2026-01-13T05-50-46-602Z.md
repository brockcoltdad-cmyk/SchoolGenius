## 2. Tone Definitions

The tone for each age group is meticulously defined to match their developmental stage and maturity level.

### K-2 (Ages 5-8)
- **Formality/Casualty**: Extremely casual, playful.
- **Energy Level**: High energy, enthusiastic.
- **Vocabulary Expectations**: Basic words (e.g., "big", "small", "share", "groups").
- **Sentence Structure**: Very short, simple sentences.
- **Words to Use/Avoid**:
  - **Use**: "Fun", "Yay", "Friend", "Let's try!"
  - **Avoid**: Complex terms like "analyze", multi-syllable words.

### 3-5 (Ages 8-11)
- **Formality/Casualty**: Casual but slightly more structured.
- **Energy Level**: Positive and encouraging, slightly less exuberant than K-2.
- **Vocabulary Expectations**: Intermediate words (e.g., "multiply", "calculate", "understand", "solve").
- **Sentence Structure**: Clear, slightly longer sentences with 2-3 steps.
- **Words to Use/Avoid**:
  - **Use**: "Great", "Challenge", "Step by step".
  - **Avoid**: Overly simplistic baby talk or advanced academic terms.

### 6-8 (Ages 11-14)
- **Formality/Casualty**: Balanced, conversational but respectful.
- **Energy Level**: Motivating, steady energy.
- **Vocabulary Expectations**: More advanced words (e.g., "analyze", "evaluate", "determine", "apply").
- **Sentence Structure**: Multi-step sentences, assumes some independence.
- **Words to Use/Avoid**:
  - **Use**: "Review", "Analyze", "Approach".
  - **Avoid**: Baby talk or overly simplistic language.

### 9-12 (Ages 14-18)
- **Formality/Casualty**: Formal, professional.
- **Energy Level**: Focused, serious but supportive.
- **Vocabulary Expectations**: Academic language (e.g., "synthesize", "deduce", "extrapolate", "demonstrate").
- **Sentence Structure**: Complex, multi-layered sentences.
- **Words to Use/Avoid**:
  - **Use**: "Systematically", "Concept", "Clarification".
  - **Avoid**: Casual slang or overly simplistic terms.

## 3. Content Adaptation by Age

Content on SchoolGenius is adapted across age groups to ensure relevance and comprehension.

- **HOW Content Changes**:
  - **Tone**: Shifts from excited and playful (K-2) to professional and academic (9-12).
  - **Complexity**: Increases from single concepts (K-2) to multi-layered critical thinking (9-12).
  - **Length**: Explanations and tasks grow longer with age, from short bursts (K-2) to detailed analyses (9-12).
  - **Examples**: Analogies become more sophisticated, e.g., fractions as "pizza slices" (K-2) vs. "rational numbers" (9-12).
  - **Steps**: Instructions range from one step (K-2) to complex multi-step processes (9-12).

- **WHAT Elements Adapt**:
  - Tone, language, complexity, sentence structure, vocabulary, emoji usage, visual design, and analogy style.
  - Content pacing and depth of explanation.

- **WHAT Stays the Same**:
  - Core educational concepts (e.g., fractions remain fractions, but explained differently).
  - Platform functionality and navigation structure.
  - Parent guides (same for all ages, no adaptation needed).

- **Specific Guidelines**:
  - Always ask: "Who is this for?" before creating content.
  - Match vocabulary to grade level (e.g., K-2: "share"; 9-12: "extrapolate").
  - Adjust complexity to cognitive capacity (e.g., K-2: one idea; 9-12: full concepts).
  - Use age-specific analogies relatable to their experiences.

## 8. Content Generation Rules

The "Pre-Implementation Checklist" ensures all content is age-appropriate before creation.

- **Checklist Before Content Creation**:
  - Does this involve user-facing content?
  - If YES, does it have 4 age-appropriate versions?
    - K-2 version (super simple, excited)
    - 3-5 version (friendly, encouraging)
    - 6-8 version (mature, respectful)
    - 9-12 version (professional, academic)
  - Are the tone, language, and complexity appropriate for each age?
  - Are emojis used appropriately for each age?
  - Are examples relatable to each age group?
  - Is the visual design age-appropriate?

- **Ensuring Age-Appropriateness**:
  - Ask: "Who is this for?" (Kindergartner, 5th grader, 8th grader, high schooler).
  - Ask: "How would they say it?" (Adjust response based on typical expressions of confusion or success).
  - Ask: "What words do they know?" (Match vocabulary to grade level).
  - Ask: "How much can they handle at once?" (Adjust steps and complexity).

- **Common Mistakes to Avoid**:
  - Using the same message for all ages.
  - Talking down to high schoolers.
  - Using complex words with young kids.
  - Overusing emojis with teens.
  - Assuming everyone knows the same things.
  - Using baby talk with middle schoolers.

## 10. Implementation Details

Technical implementation ensures age group detection and content selection are seamless.

- **getAgeGroup() Function Code**:
  ```typescript
  function getAgeGroup(gradeLevel: number | string): 'k2' | 'grades35' | 'grades68' | 'grades912' {
    const grade = typeof gradeLevel === 'string'
      ? gradeLevel.toLowerCase().startsWith('k') ? 0 : parseInt(gradeLevel)
      : gradeLevel

    if (grade <= 2) return 'k2'
    if (grade <= 5) return 'grades35'
    if (grade <= 8) return 'grades68'
    return 'grades912'
  }
  ```
