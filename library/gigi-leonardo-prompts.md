## TIPS FOR BEST RESULTS

1. **Generate base Gigi first** - Use the base prompt to create your reference image
2. **Use Character Reference** - Upload base Gigi, set to Mid strength
3. **Model choice** - Leonardo Kino XL works best for cartoon characters
4. **Batch generate** - Create 4 images at once, pick the best
5. **Iterate** - If first result isn't perfect, regenerate or adjust prompt
6. **Consistent style words** - Always include "friendly cartoon giraffe Gigi" in prompts
